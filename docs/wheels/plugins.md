## DOM to Image

缺点：

## nprogress

- [nprogress](https://github.com/rstacruz/nprogress)

超细程序进度条仿google，youtobe

```js
import NProgress from 'nprogress' 
import 'nprogress/nprogress.css' 
 
router.beforeEach((to, from, next) => {
  NProgress.start()
  next()
})

router.afterEach(() => {
  NProgress.done()
})
```
